# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities **privately** via GitHub Security Advisories:
https://github.com/0n40i4/k0nsult-uni0nai/security/advisories/new

Do not open public issues for security problems.

## Scope

This repository contains open commons (Apache-2.0). **NOT part of this repository:**
the proprietary k0nsult.cloud engine and the commercial **ipIII** module.

## Doctrine

- `claim <= proof` (evidence-first)
- **No Password Custody** — no secrets, keys, or credentials are stored in this repository
- **agents-not-people** — no scoring of natural persons
